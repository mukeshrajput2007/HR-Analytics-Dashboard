# HR Analytics Dashboard

## Overview
This project is an interactive HR Analytics Dashboard developed using Power BI. It provides insights into employee attrition, workforce demographics, and organizational trends through interactive visualizations and KPI tracking.

## Key Metrics
- Employee Count Analysis
- Attrition Count & Attrition Rate
- Attrition by Gender
- Attrition by Education
- Attrition by Age Group
- Attrition by Job Role
- Attrition by Salary Slab
- Attrition by Years at Company

## Tools Used
- Power BI
- Excel
- Data Visualization
